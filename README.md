# Beshoy Sobhy — Portfolio Website

A modern, responsive single‑page portfolio (HTML/CSS/JS) with:
- Dark/Light theme toggle
- Mobile menu
- Scroll‑spy navigation
- Projects & skills sections
- Contact form (mailto)

## 1) Quick start (local)
Just open `index.html` in your browser.

Or run a local server (recommended):
```bash
# Windows PowerShell
python -m http.server 5500
# then open http://localhost:5500
```

## 2) Customize (important)
Open `index.html` and replace:
- `USERNAME` with your GitHub/LinkedIn username
- `your.email@example.com` with your real email
- Replace `demo-link.com` and repo links with your real links
- Put your real CV file in the same folder and name it `Beshoy_Sobhy_CV.pdf`

## 3) Deploy on GitHub Pages
1. Create a repo named: `portfolio`
2. Upload these files
3. GitHub → Settings → Pages → Deploy from branch → `main` → `/root`
4. Your site will be published automatically.
